Drawing Synth

To use the synth, draw on the canvas. Different colors represent different octaves. Longer lines result in longer notes.

How it really works:

The canvas is split into eight quadrants: C, D, E, F, G, A, B, C. Four quadrants per row and two rows on the canvas. The more line data in that quadrant of the same "type" adds more time to that note. A "type" of note is the combination letter and octave. If four lines are drawn in the top left corner in a row, it will be read as one long line. The faster the line is drawn, the less data is in the quadrant.
